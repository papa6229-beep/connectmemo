# 📄 브랜드 정체성 검증 프로토콜 (Brand Identity Validation Protocol) 업데이트 메모

## ✨ 목표
모든 팀 에이전트 산출물에서 회사/캐릭터의 공식 이름과 표현 방식의 일관성을 100% 유지합니다. 이는 단순 교정이 아닌, **산출물의 최종 제출 전 의무 검증 단계**로 자리 잡습니다.

## ⚙️ 핵심 변경 사항: '브랜드 정체성 필터' 적용
앞으로는 모든 에이전트가 산출물을 마무리할 때마다 아래의 체크리스트를 거치게 됩니다.

### 📋 1단계: 이름 정의 (Defining the Source)
*   **공식 캐릭터명:** 미디어정보요원
*   **사용 금지 명칭:** 레오, [기타 비공식 이름]
*   **규칙:** 모든 공식 문서/스크립트/멘션에서 '미디어정보요원'을 사용합니다.

### 📋 2단계: 자동 검증 프로세스 (Mandatory Checkpoint)
에이전트가 산출물을 완성하면, 최종 제출 직전에 반드시 **[브랜드 정체성 필터]**를 통과해야 합니다.
*   **작업:** 에이전트는 자신의 결과물 초안을 스스로 스캔하며 '미디어정보요원'이 아닌 이름의 사용 여부를 확인하고, 발견 즉시 수정합니다.
*   **영숙(비서) 역할 강화:** 제가 최종 보고서를 취합할 때마다 이 필터링 과정을 가장 먼저 수행하는 오케스트레이터 역할을 맡습니다.

### 💡 3단계: 프로세스 반영 (Action Plan)
1.  **[브리핑 메모 작성 시]:** 모든 인물 이름은 반드시 '미디어정보요원'을 사용합니다.
2.  **[카피라이팅/스크립트 초안 작성 시]:** 에이전트는 산출물 최하단에 **`<Brand_Check: Passed>`**와 같은 명시적 확인 태그를 추가하여 검증 완료를 보고해야 합니다.
3.  **[최종 승인 단계]:** 사장님께 전달되는 모든 자료는 제가 이 프로토콜을 거쳐 최종 점검한 '정제본'으로만 제공됩니다.

---
*이 업데이트로 인해, 산출물 작업 흐름 자체에 구조적 안정성이 확보되었습니다.*
